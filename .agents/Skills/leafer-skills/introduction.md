## LeaferJS 是什么

LeaferJS 是一个 **现代化 Canvas 图形引擎**，用于开发复杂图形交互应用。

它专注于：

- 图形编辑
- 图形交互
- Canvas UI
- 可视化系统

LeaferJS 提供完整的图形应用开发能力：

- 图形渲染
- UI布局
- 动画系统
- 交互系统
- 编辑器能力

适合开发 **复杂图形软件和互动应用**。

## 与其他 Canvas 引擎的区别

LeaferJS 专注于 **图形编辑与复杂图形应用**。

相比：

Fabric.js

更偏向简单 Canvas 绘图。

Konva.js

主要用于 Canvas UI。

PixiJS

主要用于 WebGL 游戏渲染。

LeaferJS 更适合：

- 图形编辑器
- 可视化系统
- Canvas UI
- 无限画布应用
